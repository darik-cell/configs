# configs

Личные конфиги для:

- `nvim`
- `kitty`
- `IdeaVim`

Структура репозитория повторяет расположение файлов в домашней директории:

- `.config/nvim`
- `.config/kitty`
- `.ideavimrc`

Пример подключения:

```bash
ln -sfn ~/configs/.config/nvim ~/.config/nvim
ln -sfn ~/configs/.config/kitty ~/.config/kitty
ln -sfn ~/configs/.ideavimrc ~/.ideavimrc
```
